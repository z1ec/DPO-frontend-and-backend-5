## Использовано при создании работы

- HTML5 (семантическая разметка страницы)
- CSS3
- Flexbox (для построения меню навигации)
- CSS Grid (для размещения контентных блоков)
- Media Queries (адаптивная вёрстка, подход Desktop First)
- Masonry-сетка (реализована с помощью CSS, опционально)
- GitHub Pages (для публикации результата)
